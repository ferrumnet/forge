# Iron Vest Vesting Portal Backend
Install the dependencies:
```bash
sudo npm i
```

## local:
```bash
npm run nodemon-dev-api
```

## Create build:
```bash
npm run build
```
## Dev:
```bash
npm run pm2-dev-api
```

## Staging:
```bash
npm run pm2-staging-api
```

## Production:
```bash
npm run pm2-prod-api
```

App will run on 8080 port, you can update in server.ts
